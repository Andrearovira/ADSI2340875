Introdución

Se trabajo con el lenguaje Ruby y con la instalación de sus gemas como la del inicio de su sesión que cuenta con el nombre de usuario y contraseña y el de registro de usuario que se dividirá en partes como datos de personales, datos de comunicación, datos de emergencia y datos de ubicación.

Justificación

La capacitación esta para brindarle ayuda y soporte a la duda que tenga con nuestro sistema, ya que lo ayudara a llevar más control de su usuario y lo ayudara a brindar mayor información y balance del personal activo, la idea es que este sistema sea un soporte para el negocio, buscamos que le saquen el mayor provecho posible y los mejores ventajes para eso es la capacitación, así sabrán como moverse y utilizarlo.

Objetivos

Los objetivos de esta capacitación son:
  • Explicación Funcione del software.
  • El uso correcto del manejo.
  • Posibles fallas que presente y como solucionarla.
