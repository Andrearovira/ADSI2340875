1. Alcance

En este documento damos a conocer al usuario todo lo que necesita para operar en el sistema, los requisitos, funciones y demás, también la forma de manejar las distintas partes que lo componen, desde como iniciar sesión y explicación de cada una de las facetas del sistema de información, así indicándole al usuario para que sirve cada botón y hacia que sitio lo dirige, dándole así una mejor experiencia a la hora de hacer uso de nuestro sistema.


2. Requsitos para operar el sistema

Para hacer uso de este sistema se debe contar con los conocimientos básicos en la informática, necesitará de estos para el manejo del sistema, conocimientos básicos del manejo de datos personales y la base de datos, deberá conocer las políticas de seguridad, con ellas queremos asegurar a los usuarios que su información estará protegida.


3. Requisitos de Hardware-Software



Hardware un computador de mesa, un portátil y un dispositivo móvil
Software requisito mínimos sistema operativo Windows 7 memoria RAM de 4 procesador Intel(R) Pentium(R)

4. Politica de privacidad

para manejar el sistema de información deberá aceptar los términos y condiciones, estaría usted aceptando el hacer uso de sus datos personales para brindarle así mayor protección de su información y al acceder a este sistema usted reconoce que ha leído, entendido y aceptará cumplir con los términos y condiciones, teniendo usted en cuenta que el no cumplimiento puede llevar a cometer infracciones, si no está de acuerdo con lo establecido, le sugerimos abstenerse de utilizar este sitio.


5. Ingreso al sistema

Para acceder al sistema ONFIRE se encontrará con la página principal, donde habrá dos botones, uno para iniciar sección y otro para registrarse si usted no cuenta con un usuario, al presionar el botón INICIAR SECCION este lo va a dirigir a otra página.


Le aparecerá la siguiente página donde deberá digitar su usuario y contraseña, presionará el botón INGRESAR e inmediatamente usted será dirigido a nuestro sitio web


Al presionar el botón REGISTRATE este lo va a dirigir a otra página donde deberá registrar su nombre, apellido, identificación, teléfono, dirección, Eps, contacto de emergencia, prescripción médica y fecha de nacimiento, contraseña y repetir contraseña, al terminar de ingresar sus datos presionará el botón REGISTRAR y ya usted contará con un usuario y contraseña para acceder al sitio web


6. Mapa de navegacion

Acá hay una serie de botones que lo van a dirigir a diferentes sitios:

-INICIO lo llevara a donde se encontrará para iniciar sección y registrarse, también encontrara allí los planes de nutrición, planes de entrenamiento personalizados, plan de seguimiento y servicios, la ubicación, teléfono al cual contactarse y el correo electrónico.

-EVENTOS al presionarlo vera los distintos eventos que serán ofrecidos.

-CLASES vera a los diferentes entrenadores disponibles y el calendario semanal.

-SERVICIOS en esta pagina ira todo o que va a ofrecer el gimnasio como el plan de nutricion, seguimiento, entrenamientos semipersonalizados, eventos lucrativos, promociones de la tienda etc.

-TEAM ONFIRE acá estarán los miembros destacados o lider del team, calendario y rutas de team.

- ¿QUIENES SOMOS? podrá ver nuestros objetivos, misión y visión, incluyendo un mapa de ubicación.


7. Funciones de sistema

la funcion principal del sistema es mostrar el proceso de los usuarios, informar los beneficios del negocio, precios y eventos de ellos. Eso lo haremos con el inicio de sesion y el registro de usuario, los entrenadores actualizaran los procesos de los usuarios y se hara el respectivo seguimiento de cada uno con la ayuda de nuestro sistema.
