introducion y justificacion del proyecto

El propósito de nuestro proyecto tiene como finalidad la elaboración de un aplicativo web, el cual se realizará para el control de las ventas, manejo de inventario, control de los clientes, promocionar los productos de la micro empresa Dulcería el Parque. este sistema de información estará elaborado por el equipo de desarrollo.

Descripción del proyecto y entregables

Se proyecta implementar un aplicativo web que permita controlar las ganancias de las ventas, además que permita registrar todos los recorridos realizados por las unidades durante el día. Se entregó lo siguiente: • srs • Informe de análisis • Informe de diseño • Plan de Trabajo • Código fuente • Base de datos

Requerimientos del producto

Módulo de Ingreso Módulo de registro del cliente Módulo de inventario Módulo de Publicidad Módulo de Atención al cliente(PQRS) Módulo de consulta Módulo de recusación de contraseña

Requerimientos del proyecto

Satisfacer las necesidades del cliente por medio de este aplicativo web, tendrá impacto en el área administrativa del negocio ya que cambiará la forma de vender los productos y servicios que ofrece la microempresa, también hará un cambio en el control o manejo de las ventas realizadas por el trabajador y la manera como se le ofrecerán los servicios al cliente mostrándole las promociones e información de los productos y precios

alcance

Alcance Con este proyecto se quiere alcanzar la sistematización de la información de la Dulcería Parque a través de un control de ventas, un sistema de inventarios, un control de los clientes de la microempresa y mediante las redes sociales se pretende buscar incrementar la clientela, ofrecer promociones de los productos para así llevarla a conocer en la ciudad.
