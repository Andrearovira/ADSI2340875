ALCANCE

Este manual describe la funcionalidad de la aplicación “Módulo de ofertas” permite realizar la compra de productos o en su defecto realizar una búsqueda filtrada de estos mismos, para posteriormente agregarlos al carrito de compras y así finalmente realizar el pedido de estos mismos.

2.Requisitos para operar el sistema

  • Uso de navegadores
  • Conocimiento de informática básica


Requisitos de Hardware

  • Computador de minimo 4 gb de ram
  • Espacio en disco de 500 gb

Requisitos de Software
  • Sistema operativo Windows 7 o posterior, Linux kernel 3.0 o MacOS versión 10.9 en adelante.
  • Navegador Chrome versión 30 en adelante, Firefox versión 20 en adelante o Safari versión 5 en adelante.
  • Lector de archivos PDF como Acrobat Reader o similar.

4.Política de seguridad
  • Aceptar nuestras politicas de seguridad y nuestros permisos de manejo de datos
  • Crear una contraseña de minimo 8 caracteres
  • Realizar el captcha al momento del registro
  • La contraseña no debe contener caracteres especiales

Ingreso al sistema

Para acceder al aplicativo no es necesario tener un usuario ya que desde cualquier navegador podremos ingresar a él, pero al momento de realizar una compra si debemos contar con este. Para obtener un usuario debemos crearlo en el apartado de login en el cuál encontraremos un botón que nos permitirá registrarnos.
